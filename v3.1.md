# 🧠 Elder Fall Risk Prediction System v3.1（工业增强版）

---

# 🎯 1. 项目目标（最终定义）

## ✅ 系统定义

```text
输入：连续人体行为序列（3~10秒）
输出：未来跌倒风险 + 时间预测
```

---

## 🚀 双输出目标（关键升级）

```text
1. Risk Score ∈ [0,1]
2. Time-to-Fall (seconds) or No-Fall
```

---

## 📌 系统能力

| 能力   | 描述         |
| ---- | ---------- |
| 风险预测 | 当前是否接近跌倒   |
| 时间预测 | 预计多少秒后可能跌倒 |
| 趋势建模 | 风险变化曲线     |
| 多人排序 | 谁最危险       |
| 提前预警 | T-0.5~3s   |

---

# 🏷️ 7. Label System v3.1（核心重构）

---

## ❌ 错误方式（v3旧版）

```text
0.0 → SAFE
0.8 → near fall
1.0 → FALL
```

👉 人工插值，弱监督

---

## ✅ 工业标准：时间事件标签

---

## ✔ 方案 A（推荐）：Time-to-Fall Regression

```text
label = seconds_to_fall
```

示例：

| frame | label |
| ----- | ----- |
| t-3s  | 3.0   |
| t-2s  | 2.0   |
| t-1s  | 1.0   |
| fall  | 0.0   |

---

## ✔ 方案 B：Multi-horizon Risk

```text
P(fall in 1s)
P(fall in 2s)
P(fall in 3s)
```

---

## ✔ 方案 C（工业增强）：Survival Label

```text
event_time + censored data
```

```text

Shared Representation
```

## 📌 Loss函数

```text
Loss = α * MSE(risk)
     + β * SmoothL1(time)
```
