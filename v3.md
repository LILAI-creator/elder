# 🧠 8. Risk Prediction Model v3（核心）

## ❌ 不再是分类

---

## ✅ 输出：

```text
risk_score ∈ [0,1]
```

---

## 🧠 模型结构（推荐）

### ✔ GRU / TCN / Transformer

```text
Input: (T, D)
↓
Temporal Encoder
↓
Fully Connected
↓
Sigmoid
↓
Risk Score
```

---

## 📌 输出含义

| score   | 解释   |
| ------- | ---- |
| 0.0–0.3 | 安全   |
| 0.3–0.6 | 不稳定  |
| 0.6–0.8 | 高风险  |
| 0.8–1.0 | 即将跌倒 |

---

# 🧠 9. Risk Engine v3（工业级）

## 🚨 三层判断系统

---

## ✔ 1. EMA平滑

```text
s_t = α·p_t + (1-α)·s_{t-1}
```

---

## ✔ 2. 趋势分析（关键）

```text
trend = slope(last 5 frames)
```

---

## ✔ 3. 状态机

```text
SAFE → WARNING → FALL
```

---

## 🚨 判定逻辑

```text
if score > 0.4 and rising:
    WARNING

if score > 0.7 and sustained:
    FALL
```

---

# 🚨 10. 提前预警机制（核心价值）

## 🧠 为什么能提前预测？

因为跌倒是：

```text
重心下降 + 失衡 + 加速度变化
```

---

## 🚨 模型学习的是：

```text
趋势函数 f(t)
```

而不是：

```text
事件标签
```

---

## ⏱ 可提前：

| 阶段      | 时间   |
| ------- | ---- |
| 失衡开始    | T-2s |
| WARNING | T-1s |
| FALL    | T    |

---
